Nuestro trabajo:

Nuestro proyecto es un traductor de palabras a lenguaje de señas. Funciona de la siguiente forma: el usuario escribe letras o palabras y el traductor de forma rápido y clara muestra en orden las señas de las respectivas letras que el usuario escribió. Hemos hecho este proyecto para poder ayudar a los demás a comunicrse sin problemas. Esta aplicación tiene como objetivo general poder comunicarse con los sordos de manera mas sencilla y poder tener esto a donde quiera que vayas, ya que esta aplicación se puede usar tanto en cualquier computadora como en celulares Android, asi, de esta manera es mas practico. Es decir, se puede tener siempre un traductor a mano y poder usarlo de manera rápida en cualquier situación cotidiana.
